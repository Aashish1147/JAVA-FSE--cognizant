# Week 3

Work in progress.# Week 3 – Spring REST

## Overview
This week focuses on building RESTful Web Services using Spring Boot. The exercises cover creating REST APIs, handling JSON responses, retrieving data using path variables, and implementing JWT-based authentication with Spring Security.

---

## Exercises Completed

### Exercise 3 – Hello World REST Service
**Objective:**
- Create a basic REST API using Spring Boot.
- Return a simple "Hello World!!" response.

**Concepts Covered**
- Spring Boot
- REST Controller
- GET Mapping

---

### Exercise 4 – Country REST Service
**Objective:**
- Create a REST endpoint that returns Country information.
- Configure beans using Spring XML configuration.

**Concepts Covered**
- Spring REST
- Bean Configuration
- XML Configuration
- JSON Response

---

### Exercise 5 – Get Country by Country Code
**Objective:**
- Retrieve country details using a path variable.
- Return matching country information.

**Concepts Covered**
- Path Variables
- REST API
- JSON Response

---

### Exercise 6 – JWT Authentication Service
**Objective:**
- Implement authentication using Spring Security.
- Generate a JWT token after successful authentication.

**Concepts Covered**
- Spring Security
- Basic Authentication
- JWT (JSON Web Token)
- REST Authentication API
