# Exercise 5 - Cohort Tracker

## Objective

Implement styling in a React application using CSS Modules.

## Project Structure

```
src
│
├── Components
│   └── CohortDetails.js
│
├── Styles
│   └── CohortDetails.module.css
│
└── App.js
```

## Features

- Display multiple cohort details.
- Use CSS Modules for styling.
- Ongoing status is displayed in Green.
- Completed and Scheduled statuses are displayed in Blue.
- Responsive card layout.

## Technologies Used

- ReactJS
- CSS Modules

## Output

The application displays multiple cohort cards with styled information.
